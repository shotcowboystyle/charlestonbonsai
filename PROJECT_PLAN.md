## Tech Stack

| Category | Technology | Purpose |
|----------|------------|---------|
| Framework | **Nuxt 3** | Vue-based SSR/SSG framework |
| Styling | **TailwindCSS** | Utility-first CSS |
| Animation | **GSAP** | Smooth animations & scroll triggers |
| 3D Rendering | **Three.js** | Rotating tree models |
| State | **Pinia** | Global state management |
| Auth | **JWT + bcrypt** | Admin authentication |
| Database | **Supabase** | Listings & admin data |
| Hosting | **Vercel/Netlify** | Deployment |
